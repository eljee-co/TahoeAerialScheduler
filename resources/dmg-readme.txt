Tahoe Aerial Scheduler
======================

1. Make sure the Tahoe Aerials have already been downloaded in:
   System Settings > Wallpaper

2. Double-click:
   Install Tahoe Aerial Scheduler.app

3. If macOS warns that the installer is from an unidentified developer,
   right-click the app and choose Open the first time.

This installer sets up the runtime in your home folder and launches the
menu bar app automatically.
